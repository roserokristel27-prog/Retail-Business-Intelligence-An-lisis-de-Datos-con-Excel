# 📊 Retail Business Intelligence – Análisis de Datos con Excel

Proyecto de análisis de datos enfocado en la operación comercial de una red de tiendas minoristas.  
El análisis integra información de clientes, ventas, productos, tiendas y devoluciones con el objetivo de comprender el comportamiento de compra, evaluar el rendimiento comercial y generar información útil para la toma de decisiones.

---

## 🎯 Objetivo del Proyecto

Analizar la operación comercial de una red de tiendas minoristas mediante la integración de distintos datasets, con el fin de:

- Identificar patrones de consumo de los clientes
- Analizar el desempeño de ventas
- Evaluar devoluciones de productos
- Generar indicadores clave para la gestión comercial

---

## 🛠️ Herramientas utilizadas

- Microsoft Excel
- Tablas estructuradas
- Funciones de Excel
- Tablas dinámicas
- Gráficos para visualización de datos

---

## 🔎 Proceso de análisis

El proyecto se desarrolló en tres etapas principales.

### 1. Preparación y limpieza de datos

- Importación de datasets en Excel
- Formateo de columnas según tipo de dato
- Limpieza de valores faltantes
- Organización de datos en tablas estructuradas
- Creación de rangos y cálculos iniciales

---

### 2. Enriquecimiento de datos

- Integración de datasets mediante funciones de búsqueda
- Identificación y tratamiento de valores nulos
- Creación de variables derivadas
- Cálculo de indicadores estadísticos (máximo, mínimo, promedio y mediana)
- Estandarización de variables y limpieza de datos

---

### 3. Visualización y análisis dinámico

- Creación de tablas dinámicas
- Análisis de ventas por región
- Análisis de devoluciones por motivo
- Identificación de clientes con mayor gasto
- Identificación de productos con mayor número de devoluciones

---

## 📊 Resultados obtenidos

A partir del análisis de los datos se identificaron diversos patrones relevantes para la operación comercial, entre ellos:

- Regiones con mayor volumen de ventas
- Clientes con mayor nivel de gasto
- Productos con mayor frecuencia de devolución
- Canales de venta con mayor participación en las ventas totales
- Tendencias en el comportamiento de ventas a lo largo del tiempo
